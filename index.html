goauth by alloy-d

A Go library for doing header-based OAuth over HTTP or HTTPS. Mostly created for working with Twitter.

Dependencies

[Go](http://golang.org/)

Install

goinstall github.com/alloy-d/goauth

$ git clone git://github.com/alloy-d/goauth
